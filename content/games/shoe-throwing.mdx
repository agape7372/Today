## 준비

- 표적을 거리별 3-5개 배치 (1m·2m·3m, 점수 차등)
- 출발선 표시 — 환자별 거리 조정 가능
- 신발 1인당 2-3개 배분
- 시범 1회 — 신발 던지기 동작 (어깨 → 손목 흐름)

## 진행 단계

1. **워밍업:** 가까운 표적(1m)에 가볍게 5번
2. **라운드 1 — 정확도:** 모든 거리 시도, 점수 합산
3. **라운드 2 — 환측 손만:** 편마비는 환측 단독 사용
4. **라운드 3 — 팀 대결:** 두 팀 점수 비교, 응원 분위기
5. **마무리:** 최고점 환자 박수 + 회상 대화 ("옛날에 신발 던지기 해본 적 있나요?")

## 변형

- **난이도↑:** 표적 작게, 거리 확장, 시간 제한
- **난이도↓:** 큰 표적, 짧은 거리, 양손 자유
- **CIMT:** 건측 미틀 착용, 환측 단독
- **휠체어:** 무릎 위에서 굴려서 표적으로
- **회상 강조:** "옛날 추억" 대화 + 신발 모양·색 다양화

## 안전 주의

- 가벼운 실내용 신발만 (운동화 X)
- 던질 때 사람 향하지 않게
- 좌석 안정성 점검
- 호흡 거칠 시 즉시 휴식

## 평가 포인트

- 라운드별 점수 (정확도)
- 환측 손 성공률 (편마비 추적)
- 회상 대화 참여 (사회적·인지적 지표)
- Box & Block Test 사전·사후

> SC운동발달아카데미 (목포 노인체육) — 어르신에게 친숙한 일상 소품(신발)을 활용해 거부감 0 + 회상 자극 + 던지기 task 통합.
