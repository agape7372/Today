## 준비

- 색깔 콘을 원형 또는 일자 배치 (간격 2m)
- 색깔 신호판 또는 카드 코치 손에
- 환자에게 색-동작 약속 설명 (예: 빨강 = 멈춤, 초록 = 차기, 노랑 = 줍기)
- 시범 1회 — 색 보이고 동작 수행

## 진행 단계

1. **워밍업:** 2가지 색만 사용 (1분) — 색-동작 약속 익히기
2. **라운드 1 — 4가지 색:** 코치 신호에 따라 동작 수행, 정확도 점수
3. **라운드 2 — 속도 증가:** 신호 빠르게 — 반응속도 도전
4. **라운드 3 — 반대 명령:** 빨강 = 차기, 초록 = 멈춤 (인지 전환 부하)
5. **마무리:** 자유 진행 — 즐거움 강조

## 변형

- **난이도↑:** 색 수 증가, 빠른 신호, 반대 명령 라운드 추가
- **난이도↓:** 2가지 색만, 천천히
- **휠체어:** 앉아서 공 차기·줍기로 변형
- **파킨슨:** 명확한 시각·청각 단서 동시 — 신호판 + 코치 구호
- **편마비:** 환측 발 사용 강제 라운드

## 안전 주의

- 빠른 방향 전환 시 낙상 모니터링
- 보조자 1-2명 동반
- 호흡 거칠 시 즉시 휴식
- 15분 초과 시 피로감 점검

## 평가 포인트

- 신호 인식 정확도 (색 변별)
- 반응 시간 (이중과제 처리)
- 환측 발 사용 비율 (편마비)
- TUG·Stroop Test 사전·사후 비교

> Petzinger(2013) — 신체 운동 + 인지 과제 결합이 파킨슨의 신경가소성을 단순 운동보다 강력히 촉진. 색 신호 축구는 운동-인지 이중과제의 모범 예.
