## 준비

- 분류 그릇 5-6개를 색별로 배치, 색 라벨 부착
- 출발 트레이에 색 카드·칩 무작위 섞기
- 환자 자세 — 손목 중립, 시야 30cm
- 색 이름 시범 1회

## 진행 단계

1. **워밍업:** 2가지 색만 분류 (1분)
2. **라운드 1 — 4가지 색:** 무작위 카드 한 장 → 색 호명 → 그릇 분류
3. **라운드 2 — 5가지 색·속도:** 같은 과제 시간 단축
4. **라운드 3 — 환측 손만:** 편마비는 환측 단독
5. **인지 결합:** 색 → 그 색의 사물 호명 (예: 빨강 = 사과·소방차)

## 변형

- **난이도↑:** 색 수 증가, 속도 도전, "같은 색이지만 명도 다른" 카드 추가
- **난이도↓:** 2-3색만, 큰 카드, 양손 자유
- **CIMT:** 건측 미틀
- **파킨슨:** 큰 동작 강조 — 카드를 멀리 그릇에 던지듯
- **회상 자극:** "이 색 보면 뭐 생각나세요?" 대화

## 안전 주의

- 작은 칩 삼킴 모니터링
- 색맹 환자는 색 + 모양 병행
- 15분 초과 시 안구 피로
- 손목 신전 자세 5분마다 휴식

## 평가 포인트

- 라운드당 분류 카드 수 (속도·정확도)
- 오분류 비율 (시각 변별)
- 환측 손 사용 비율 (편마비)
- Stroop Test 사전·사후 비교 권장

> 기존 색·숫자 분류(`color-number-sort`)가 색+숫자 이중 단서라면, 이 게임은 **색만 + 회상 + 사물 연결**로 확장. 인지장애 초기·중등도에 더 적합.
