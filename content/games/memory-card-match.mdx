## 준비

- 환자 인지 수준에 맞춘 카드 쌍 수 결정 (4쌍부터 시작)
- 카드 뒤집어 4×4 또는 4×6 격자 배치
- 휠체어 환자는 테이블 높이 확인 (팔꿈치 90°)
- 좌석 안정성·조명 점검

## 진행 단계

1. **라운드 1 — 탐색:** 모든 카드 30초 공개, 위치 암기
2. **라운드 2 — 시도:** 차례로 카드 2장 뒤집기, 짝이면 획득·아니면 재덮기
3. **협력 모드:** 다른 환자의 단서 허용 ("거기 그 카드!"), 단 결정은 본인
4. **라운드 3 — 점수:** 개인이 아닌 **팀 총점**으로 발표 (협력 강조)

## 변형

- **난이도↑:** 카드 수 증가, 노출 시간 단축, 색·패턴 동시 매칭
- **난이도↓:** 4쌍 시작, 노출 시간 60초, 짝지은 사람 카드 즉시 제거
- **편측 무시:** 환측 시야에 단서 카드 의도적 배치 (CIMT 응용)
- **파킨슨:** 카드 뒤집기 직전 명확한 시작 신호 추가

## 안전 주의

- 좌석 안정성 점검 (휠체어 브레이크, 의자 미끄럼)
- 손가락 관절염 환자는 손바닥 전체로 카드 잡기
- 30분 초과 시 피로감 모니터링

## 평가 포인트

- 라운드별 짝 맞춘 수 (단기기억 추세)
- 단서 제공 빈도 (사회적 상호작용 지표)
- 종료 후 자가 평가 1-5점 (재미도)

> Sweller(1988) 인지부하 이론 — 카드 수는 작업기억 한계(4±1 청크) 이내 시작.
